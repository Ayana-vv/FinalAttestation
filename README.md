### Итоговая аттестация

**Реквизиты:**
* ui-тесты: https://www.saucedemo.com/
* api-тесты: https://innopolispython.onrender.com/apidocs/#/

**Во фреймворк включены:**
- работа с DB
- работа с API
- работа с UI

**Используемые инструменты:**
Для выполнения работы в качестве инструментов использованы следующие библиотеки:

* JDBC - для подключения приложения к базе данных.;
* RestAssured - библиотека для тестирования REST API на Java;
* JUnit 5 - фреймворк для запуска тестов;
* Selenium - уинструмент для автоматизации браузеров, позволяющий писать UI-тесты;

**Запуск тестов из терминала:**
`mvn test`
